Code Summary

For transparency and so even those with no site building knowledge can use this tool to add a plant to the site below is a simple explanation on what the html this app generates means. This does not add anything to the site directly just creating the code for you to use with a simple copy/paste to add it to the site manually.

  • section - One of the standard container elements for different sections of a page.

  • h3 - Heading, ranging in importance from most at h1 to least at h6. Important for navigation, particularly for those using tools like a screenreader.

  • p - Paragraph, the standard element for text on a site.

  • ol and ul - Ordered (numbers) and Unordered (bullet points) list, can disable the numbers/bullets if using just for formatting and positioning. Also makes site easier to navigate.

  • li - List item, placed inside a ol or ul element.

To add to site enter the plant information in the form, click submit, the code will be automaticly saved to your clipboard ready to be pasted. After that open seedlist.html in whatever way you used to edit the site and scroll down til reach the right alphabetical section for it and paste between the "ul" elements in the proper alphabetical section. All the styling will be applied automaticly.

To make navigating easier each seedlist section has an id along lines of "section__a", "section__m", or "bulky__section__a". So can just easily ctrl+f like would for anything text.

Click button for Form to switch between cultivar form (IE: Hon Tsai Tai) or plant group form, IE: Asian Greens,
